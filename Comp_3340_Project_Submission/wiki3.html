Using The Exercise API

Within the Exercise Selection Tool page, a user can search for a specific exercise that they are interested in performing in the search field. Then, by clicking on the "Search" button, the page will be populated with a variety of different versions of that exercise, as well as the equipment required to complete it. As a bonus, this page links to our Exercise Library, where we host videos of the best tips and tricks for performing exercises properly and safely.
